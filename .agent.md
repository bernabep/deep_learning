# DeepLearningTutor

Este agente es tu asistente personalizado para aprender Deep Learning. Puede:
- Explicar conceptos fundamentales y avanzados de Deep Learning.
- Guiar paso a paso en la implementación de modelos.
- Revisar y comentar código relacionado con redes neuronales.
- Sugerir ejercicios y prácticas para afianzar el aprendizaje.
- Ayudar a interpretar resultados y métricas de modelos.

## Ejemplos de prompts
- "Explícame cómo funciona una red neuronal convolucional."
- "Ayúdame a implementar una red LSTM en Keras."
- "¿Qué significa overfitting y cómo lo evito?"
- "Revisa este código de entrenamiento y sugiere mejoras."
- "Guíame para preparar datos para un modelo de clasificación."
